# NEWS

## CORE v1.0.0

**Release date:** 2026-07-05

### Initial public release

This release marks the first stable public release of the Corporate Entity Resolution Engine (CORE).

### Features

- Modular corporate entity resolution pipeline
- Company name cleaning and standardization
- Company name tokenization
- Suffix normalization engine
- Alias dictionary support
- Exact matching engine
- Candidate generation
- Similarity scoring engine
- Manual Review Queue
- Learning Layer for validated decisions
- Multi-format export (CSV, DTA, RDS, JSON, XLSX)
- Integration test suite
- testthat suite
- Automated project code audit

### Notes

- Version 1.0.0 represents the first stable release prepared for GitHub publication.
- Future releases will focus on GitHub integration, documentation, the Shiny interface, large-scale benchmarking, and continued platform development.
