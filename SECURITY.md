# Security Policy

## Reporting a Security Issue

If you believe you have identified a security vulnerability in CORE, please report it by email rather than creating a public GitHub issue.

**Contact**

Yurtsev Uymaz  
y.uymaz@uea.ac.uk

Please include:

- a description of the issue,
- steps required to reproduce it,
- the affected module(s),
- your operating system and R version, if relevant.

Reports will be acknowledged as soon as possible and investigated before any public disclosure where appropriate.

## Scope

CORE is an academic research software project intended for transparent and reproducible entity resolution. Security reports related to source code, dependencies, or distributed releases are welcome.
