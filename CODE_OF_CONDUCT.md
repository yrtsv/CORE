# Code of Conduct

## Our Commitment

CORE is an academic research software project. We are committed to maintaining a respectful, constructive, and inclusive environment for all contributors, users, and researchers.

## Expected Behaviour

Participants are expected to:

- communicate respectfully,
- provide constructive feedback,
- respect different levels of technical experience,
- focus criticism on ideas, code, and documentation rather than individuals,
- support transparent and reproducible research practices.

## Unacceptable Behaviour

Unacceptable behaviour includes:

- harassment or intimidation,
- discriminatory or exclusionary language,
- personal attacks,
- persistent disruption of discussions,
- sharing private information without permission.

## Scope

This Code of Conduct applies to all CORE project spaces, including GitHub issues, pull requests, documentation discussions, and related project communication.

## Purpose

The purpose of this Code of Conduct is to support a productive research software environment in which contributors can improve CORE through careful review, open discussion, and reproducible scientific practice.
